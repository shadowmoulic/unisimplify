# UniSimplify

UniSimplify is a comprehensive platform designed to streamline the university application and discovery process for students.

## 🚀 Features

- **Discover Universities**: Explore a curated list of universities and programs.
- **Student Dashboard**: Manage your applications, saved colleges, and track your progress.
- **Contact & Support**: Get in touch with experts for guidance.

## 🛠️ Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS / Vanilla CSS
- **Language**: TypeScript

## 💻 Getting Started

First, install dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `/web/src/app` - Main application routes and pages
- `/web/src/app/discover` - University discovery module
- `/web/src/app/dashboard` - Student dashboard module
- `/web/src/app/contact` - Contact page
